# jgrazian.github.io

1. Clone
    ```
    git clone https://github.com/jgrazian/jgrazian.github.io.git
    cd jgrazian.github.io
    ```

2. Serve
    ```
    hugo server
    ```

3. Build
    ```
    hugo
    ```

- To build Javascript from ./src.
    ```
    tsc
    ```

- To see drafts
    ```
    hugo server -D
    ```
